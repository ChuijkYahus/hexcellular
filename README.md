# Hexcellular

Adds a fairly unbalanced form of iota storage called property iotas. These are essentially focuses inscribed into the fabric of the universe itself; they can be written to for very cheap and read from freely.

They can be used in many applications like avoiding the stack limit in long Hexes by offloading branches into properties, or to communicate between player-cast Hexes, casting devices, wisps, spell circles, and anything else that have an iota embedded. It can also be used to retain state, for example a casting device that increments a property in order to know how many times it has been cast and do something different each time.
